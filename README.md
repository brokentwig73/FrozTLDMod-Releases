# FrozTLDMod

Public release downloads and documentation for **FrozTLDMod**, a collection of quality-of-life improvements for *The Long Dark*.

The source code is maintained in a private repository. This repository contains only release packages, installation instructions, and public issue tracking.

## Requirements

- The Long Dark: Survival Mode on Windows
- MelonLoader 0.7.2
- ModSettings

## Installation

1. Install MelonLoader and ModSettings.
2. Download the latest `FrozTLDMod.zip` from the Releases page.
3. Extract the archive into the game's `Mods` folder.
4. Confirm these files exist:
   - `Mods/FrozTLDMod/FrozTLDMod.dll`
   - `Mods/FrozTLDMod/manifest.json`
5. Launch the game and configure **FROZ TLD MOD** from Mod Settings.
